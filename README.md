## Tech Stack

- HTML5 semantic markup
- CSS3 (Grid, Flexbox, animations, keyframes)
- Vanilla JavaScript (IntersectionObserver, event listeners)
- No build tools required
- Mobile-first responsive design

## Design Philosophy

- **Above the fold matters:** 57% of visitor attention in first 5 seconds
- **Progressive enhancement:** Animations enhance, content works without JS
- **Accessibility:** Semantic HTML, color contrast, keyboard nav, prefers-reduced-motion
- **Data-driven:** Conversion optimization based on UX research

## How to Use These Designs

Each project is a **standalone HTML file** (HTML/CSS/JS in one or split into 3 files). Open in any browser, no build step needed.

For local development:
1. Clone this repo
2. Open any `index.html` in your browser
3. Modify `style.css` or `script.js` as needed
4. Push to GitHub — GitHub Pages auto-publishes
