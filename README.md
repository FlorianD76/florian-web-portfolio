# Florian Dierckx — Web Portfolio

Web design & development. Demonstrating modern, responsive web experiences for B2B and e-commerce.

## Projects

### BEVAC Engineering — Redesign

Two versions showcasing design evolution:

- **[V2 — Technical Premium](./bevac/v2/)** — Sober, technical direction. Ideal for B2B engineering firms. Clean grids, proof strip, direct CTA.
- **[V3 — Modern + Animations](./bevac/v3/)** — Same technical direction with enhanced motion. Animated counters, reveals, spotlight effects, grid drift.

**The Challenge:**  
60 years of engineering expertise (Brussels Airport, STIB, EU Commission) was invisible to prospects. First 5 seconds = make it obvious.

**The Solution:**  
Dark technical aesthetic, proof-first structure, signal blue + amber accent, Clash Display typography. V2 establishes credibility. V3 adds polish for modern audiences.

**Why It Works:**  
- Puts social proof above the fold (clients you recognize)
- No fluff, no vanity copy — just credentials + CTA
- Responsive grid adapts to any device
- Animations respect user preferences (prefers-reduced-motion)

### Angry Dollz — E-Commerce

**[View](./angry-dollz/)** — Exclusive bikinis, lingerie, custom pieces. Limited-edition fabrics, pole/aerial/stage wear.

**The Challenge:**  
Bold brand, niche audience, need to signal exclusivity + quality without excessive copy.

**The Solution:**  
Y2K doll aesthetic (black + hot pink chrome), animated category cards, pole dance SVG animation section, scarcity badges (Limited, New), Instagram community integration.

**Why It Works:**  
- Color + motion immediately signals the vibe (attitude, playfulness, premium)
- Pole dance animation is a differentiator (most e-commerce sites don't have animations this specific)
- Simplified navigation (4 categories, not 20 filters)
- Social proof via Instagram integration (community > testimonials)

## Tech Stack

- **HTML5** — Semantic markup for SEO and accessibility
- **CSS3** — Grid, Flexbox, animations, keyframes
- **Vanilla JavaScript** — IntersectionObserver for reveals, event listeners, no frameworks
- **Google Fonts + Fontshare** — Clash Display, Satoshi, Space Grotesk
- **No build tools** — Code is readable, maintainable, no dependencies
- **GitHub Pages** — Deploy from repo, auto-publish on push

## Design Philosophy

**Above the fold matters:** 57% of visitor attention in the first 5 seconds. Every pixel counts.

**Progressive enhancement:** Animations enhance, content works without JS. Accessibility first.

**Data-driven:** Designs based on UX research, not trends. Conversion optimization > aesthetics.

**Responsive by default:** Mobile-first design. Tested at 375px, 680px, 920px, 1280px+.

## Performance

- No build step required
- No JavaScript frameworks (lighter, faster)
- Minimal dependencies (only Google Fonts CDN)
- Static HTML/CSS/JS (instant loading)
- GitHub Pages hosting (free, fast, reliable)

## How to Use These Designs

Each project is **standalone and self-contained**. You can:

1. **Learn from the code** — Open the HTML, CSS, JS files and study the patterns
2. **Clone and customize** — Fork the repo, modify colors/copy/images for your business
3. **Use as a template** — Adapt the structure for your own projects

### For Local Development

```bash
# Clone the repo
git clone https://github.com/florian-dierckx/florian-web-portfolio.git
cd florian-web-portfolio

# Open any project in your browser
open bevac/v2/index.html
open angry-dollz/index.html

# Edit and test locally
# CSS changes auto-reload in the browser
# JavaScript changes require a page refresh
```

### For Deployment

```bash
# Push to GitHub — GitHub Pages auto-publishes
git add .
git commit -m "Update design"
git push origin main

# Site updates at https://floriand76.github.io/florian-web-portfolio/
```
