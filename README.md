## Projects

### BEVAC Engineering — Refonte de site web
**Two versions showcasing design evolution:**

- **[V2 — Technical Premium](./bevac-v2/)** — Sober, technical direction. Ideal for B2B engineering firms. Features hero with proof strip, bento service cards, domain grid, testimonial section.

- **[V3 — Modern + Animations](./bevac-v3/)** — Same technical direction with enhanced motion. Animated counters, word-by-word reveal, tile spotlight effects, grid drift. Demonstrates progressive enhancement.

**Project Details:**
- Client: BEVAC Engineering SPRL (Brussels)
- Brief: Redesign homepage to highlight 60+ years of expertise and key references (Brussels Airport, STIB, EU Commission)
- Challenge: Transform invisible expertise into a first-5-second proof statement
- Design approach: Dark technical aesthetic, grid backgrounds, signal blue + amber accent, Clash Display typography

---

## Tech Stack

**Frontend:**
- HTML5 semantic markup
- CSS3 (Grid, Flexbox, animations, keyframes)
- Vanilla JavaScript (IntersectionObserver, event listeners)
- Google Fonts + Fontshare custom typefaces

**Animations & Effects:**
- Reveal on scroll (IntersectionObserver)
- Count-up numbers (requestAnimationFrame)
- CSS keyframes (hero word-rise, pole dancer spin, drift effects)
- Hover states (3D tilt, spotlight follow, glow effects)

**Responsive Design:**
- Mobile-first approach
- Tested at 375px, 680px, 920px, 1280px+ breakpoints
- Touch-friendly interactions

## Design Principles

- **Above the fold matters:** 57% of visitor attention in first 5 seconds
- **Mobile-first:** 50%+ of traffic is mobile; thumb-zone design
- **Progressive enhancement:** JS adds polish, content works without it
- **Data-driven:** Conversion optimization based on UX research
- **Accessibility:** Semantic HTML, color contrast, keyboard navigation
