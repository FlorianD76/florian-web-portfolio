# BEVAC Engineering — V2 (Technical Premium)

**Live:** https://florian-dierckx.github.io/bevac-v2/

A sober, technical redesign for a 60-year-old Belgian engineering bureau specializing in railway signalling and transport safety.

## Design Direction

- **Dark technical aesthetic** — grid backgrounds, signal blue + amber accents
- **Clash Display typography** — bold, affirmative display font
- **Hero-first structure** — immediate clarity on what they do + proof strip
- **Trust signals** — named clients (Brussels Airport, STIB, EU Commission) visible above fold
- **B2B conversion focus** — clear CTAs, no friction, RAMS/FDMS credentials front-and-center

## Key Sections

1. **Hero** — "La sécurité des transports, sans compromis" + System Readout panel (60+ years, 100% independent)
2. **Proof Marquee** — scrolling client logos
3. **Services** — 6 bento cards (Study, Relecture, Audit, Formalization, Project Management, Network Management)
4. **Domains** — 8 application areas (Railway Signalling, Airport Beaconing, Metro/Tram, etc.)
5. **Approach** — Why independence matters + mission quote
6. **References** — grid of 12 official partners (UITP, CENELEC, UIC, Commission EU, etc.)
7. **CTA Band** — final conversion moment with trust line

## Files

- `index.html` — semantic markup, all sections
- `style.css` — dark theme, grid backgrounds, responsive layouts
- `script.js` — header scroll state, reveal on scroll, mobile menu

## Performance

- Single-page, no build tools required
- Fonts loaded from Fontshare CDN
- ~29KB HTML (unminified for clarity)
- Mobile-responsive at 375px+

## Notes for Clients

This version is **professional and serious**, ideal for:
- Engineering firms, infrastructure companies
- Government/institutional buyers
- Technical audiences who value precision over flash

For a more modern, animated version, see **V3**.
